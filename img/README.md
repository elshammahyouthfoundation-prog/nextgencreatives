# Images Folder / Folda ya Picha

This folder contains all images used in the NextGen Creatives Tours website.

## Purpose / Madhumuni

- Store website images (picha za tovuti)
- Tour destination photos (picha za maeneo ya ziara)
- Gallery images (picha za mkusanyiko)
- Icons and graphics (ikoni na michoro)

## Usage / Matumizi

Place all image files in this folder and reference them in your code as:
```
img/your-image-name.jpg
```

Weka faili zote za picha katika folda hii na zitaje katika code yako kama:
```
img/jina-la-picha-yako.jpg
```

## Image Guidelines / Miongozo ya Picha

- Use descriptive filenames (tumia majina yanayoelezea)
- Optimize images for web (boresha picha kwa ajili ya wavuti)
- Supported formats: JPG, PNG, GIF, SVG, WebP
